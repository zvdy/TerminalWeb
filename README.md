## Show commands

To show the commands, you can use the `help` command inside the terminal.

## Commands

### `help`

Show the list of commands.

### `clear`

Clear the terminal.

### `whois`

Show the information about the author.

### `whoami`

Show the information about the user.

### `sociial`

Show the social media links.

### `secret`

Show the secret of the terminal.

### `projects`

Where to see my projects and code.

### `history`

Show the history of the terminal.

### `email`

Send email to author.

### `banner`

Show the banner of the terminal.

### `uname`

Show the information about the system.

### `exit`

Exit the terminal.
